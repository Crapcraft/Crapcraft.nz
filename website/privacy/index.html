Privacy Policy

1. Who We Are

crapcraft.nz is a privately operated hobbyist platform hosted in New Zealand. It is not a business, generates no revenue, and is run by a single individual for personal use and a small friend group.

By using any service under the crapcraft.nz domain, you agree to the terms described here.

2. What We Collect

The following is a summary of what data each service may collect:

All Services: IP address, timestamps, HTTP request metadata (browser, OS, path) - server access logs for security, abuse prevention, and debugging.

Authentik: Username, email address, hashed password, login timestamps, session tokens - used for account authentication and access control.

Jellyfin: Username, playback history, media preferences, device info - used for media streaming functionality and watch history.

Immich: Photos and videos you upload, EXIF metadata (may include GPS location, camera info, timestamps) - used for photo library hosting, stored only for your use.

Seerr: Username, media request history, linked Jellyfin account - used for media request management.

Minecraft: Minecraft username, IP address, in-game activity and chat messages - used for server operation, moderation, and anti-cheat.

Discord Bots: Discord user ID, message content when commands are triggered, server/channel IDs - used for bot functionality, not stored long-term unless a feature requires it.

Map: No personal data collected - a public Minecraft world map viewer, no login required.

FMD: Device location, battery status, and push commands (all end-to-end encrypted - not readable by us), used for Find My Device functionality. FMD also maintains its own privacy policy.

Notes: Documents and content you create, username via Authentik SSO - used for note-taking and document storage.

Vaultwarden: Encrypted vault data (client-side encrypted - not readable by us), account email, IP address - used for password manager functionality.

ntfy: Topic subscriptions, notification delivery metadata - used for push notifications for platform services.

SearXNG: Search queries (not logged or stored beyond the request) - used for privacy-respecting search aggregation.

Open WebUI: AI conversation history, username via Authentik LDAP - used for the AI chat interface.

Tracearr: Username, media tracking data, watch history and statistics - used for media activity tracking and statistics.

Uptime Kuma: No personal data collected via the public status page - service uptime metrics only.

FMHY Docs: Server access logs only (IP, timestamp, user-agent) - no accounts required, used for a static documentation site.

Zipline: Files uploaded by the administrator, access timestamps, IP addresses of viewers - used as a CDN for hosting publicly viewable files (images, video, audio, etc.); upload access is restricted to the administrator.

AzuraCast: Listener IP address, listener agent, listening duration (for the public radio stream), plus account credentials and station config for admin access - used for internet radio streaming and public stream listener metrics.

slskd: IP address, shared file listings, transfer logs (IPs of peers who download from us).Used for P2P file sharing (Soulseek protocol).

Media Automation (Sonarr, Radarr, Prowlarr, Bazarr, qBittorrent):Access is gated by a reverse-proxy login (Authentik) placed in front of them. Each tool separately logs search and download history and indexer API interactions - used for automated media acquisition and management; access is restricted to authorised users.

IP addresses are logged as part of normal server operation for security purposes. They are never sold, shared with third parties, or used for advertising. This notice satisfies the indirect collection requirement under IPP 3A of the Privacy Amendment Act 2025.

Note on encrypted services: FMD and Vaultwarden use client-side or end-to-end encryption. The server stores encrypted data only - the administrator cannot read the contents under any circumstances.

3. How We Use Your Data

Data is used solely to operate these services - to authenticate you, deliver functionality, identify abuse, and diagnose bugs.

Your data is never sold, used for advertising, shared with third parties, or processed for any commercial purpose. Full stop.

4. Where Data is Stored

All data is stored on privately owned hardware in New Zealand. No third-party cloud services (AWS, Google Cloud, Azure, etc.) are used for storage.

Network traffic passes through Cloudflare for DDoS protection and tunnelling - they may process IP addresses and request metadata in transit. See Cloudflare's Privacy Policy.

Some outbound traffic is routed through ProtonVPN. See ProtonVPN's Privacy Policy.

5. Your Rights (NZ Privacy Act 2020)

Under the New Zealand Privacy Act 2020 and the Privacy Amendment Act 2025, you have the right to access, correct, or delete any personal information held about you, and to know how it is being used.

The Privacy Amendment Act 2025 introduced IPP 3A (in force from 1 May 2026), which requires that when personal information is collected indirectly - such as IP addresses and server logs - you are notified of that collection. This privacy policy serves as that notification.

Contact the administrator using the details below. Requests will be responded to within 20 working days, as required by the Act.

6. Data Retention

Server access logs: retained for 7 days for security and debugging purposes, then automatically cleared.

Account data: retained for the lifetime of your account, deleted on request.

Immich media: retained until you delete it or your account is removed.

Zipline files: retained until removed by the administrator.

Notes content: retained until you delete it or your account is removed.

Vaultwarden vault: retained until you delete it or your account is removed. Contents are encrypted and unreadable by the administrator.

Minecraft logs: retained for a limited period for moderation purposes.

Map: no personal data stored.

Discord bot interactions: not persistently stored unless a feature explicitly requires it.

SearXNG queries: not stored beyond the duration of the request.

Open WebUI conversations: retained in your account until deleted.

Tracearr tracking data: retained until deleted or your account is removed.

Uptime Kuma: no personal data is stored or retained.

FMHY Docs: server access logs only, retained for a limited period then cleared.

AzuraCast: listener logs retained for a limited period, then cleared. Station config retained for the lifetime of the service.

slskd: transfer logs retained for a limited period. Shared files retained until removed by the administrator.

Media automation data: search and download history retained for the lifetime of the service. Can be cleared on request.

These services are not intended for anyone under the age of 13. If you are under 13, do not use these services or provide any personal information.

Cookies & Logs

1. Cookies

Only functional cookies are used - strictly necessary for services to operate. No tracking, advertising, or analytics cookies.

Session token (Authentik): keeps you logged in across services. Duration: session / configurable.

Media session (Jellyfin): remembers your login and preferences. Duration: session.

Vault session (Vaultwarden): keeps you logged in to the password manager. Duration: session / configurable.

__cf_bm, cf_clearance (Cloudflare): DDoS protection and bot detection. Duration: varies.

No cookie consent banner is shown because only strictly necessary functional cookies are used. If that changes, a consent banner will be added.

2. Server Access Logs

All HTTP requests to services under crapcraft.nz are logged by the web server. Each entry typically includes your IP address, date and time, the URL requested, HTTP status code, and your browser's User-Agent string.

Logs are stored locally and not shared externally. They are retained for a week for security purposes and then cleared.
